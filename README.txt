## DESCRIÇÃO DO CONTEUDOS DE CADA PASTA ##

ACTIONS -> PASTA ONDE FICARÃO AS ACTIONS (REDUX);
COMPONENTS -> PASTA ONDE FICARÃO OS COMPONENTES REUTILIZAVEIS (EX: INPUTS, BUTTONS);
DOMAINS -> PASTA MODEL (CRIAÇÃO DAS CLASSES/OBJETOS);
PAGES -> PASTA VIEW, DESTINADA A TODO O FRONT END (PAGES) DA APLICAÇÃO;
REDUCERS -> PASTA ONDE FICARÃO OS REDUCERS (REDUX) ;
UTILS -> PASTA ONDE FICARÃO OS METODOS GENERICOS, COM UTILIDADES DE USO FREQUENTE;
SCRIPTS_JSON -> PASTA ONDE FICARÃO OS CODIGOS EM JSON DAS "TABELAS" DO FIREBASE;


Init.js -> CLASSE DE INICIALIZAÇÃO DA APLICAÇÃO;
Navigation.js -> CLASSE DE NAVEGAÇÃO ENTRE AS PAGES DA APLICAÇÃO;
dependencies.txt -> ARQUIVO ONDE CONTEM TODAS AS BIBLIOTECAS E SUAS VERSÕES UTILIZADAS NA APLICAÇÃO;
script_tabelas_bd.json -> ARQUIVOS ONDE CONTEM "TABELAS" DO BANCO;
